ISAT B321 Project Instructions: 


1. Run the B321_Team02_1_CreatePopulate.sql file first on SSMS and make sure that all of the data is populated. 

2. Run the B321_Team02_2_TSQL.sql file second on SSMS and make sure that all of the stored procedures are created. 

3. Run the B321_Team02_3_CreateAccounts.sql if necessary, to create the database users and assign all of the necessary permissions.

4. Unzip the B321_Team02_VSSolution.sql file and run the B321_Final_Project_.NET.sln file. 

5. Change the connection string if needed to run the project.
